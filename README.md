# Banco-de-dados-Azure

# APLICANDO PASSO A PASSO COMO CRIAR UMA

* JÁ UTILIZANDO O PORTAL DO AZURE VAMOS CRIAR O RECURSO:
  - VAMOS EM "CRIAR UM RECURSO"
  - BANCO DE DADOS -> INTÂNCIA GERENCIADA DO AZURE SQL
  - 
* CONFIGURANDO E DEFININNDO A INSTÂNCIA:
  - NOME DA INSTÂNCIA (NOME ÚNICO).
  - REGIÃO (LOCALIZAÇÃO DO DATACENTER).
  - GRUPO DE RECURSOS (PODE CRIAR UM NOVO OU UTILIZAR UM EXISTENTE).
  - LOGIN DE ADMINISTRADOR E SENHA.

* CONFIGURANDO A REDE
  - É NECESSÁRIO ASSOCIAR A INSTÂNCIA A UMA REDE VIRTUAL (VNET).
  - CONFIGURE SUB-REDES E REGAS DE ACESSO PARA PERMITIR CONEXÕES SEGURAS.
 
* ESCOLHENDO O DESEMPENHO E ARMAZENAMENTO
  - SELECIONE O MODELO DE COMPRA (vCore).
  - DEFINA O NÚMERO DE NÚCLEOS, MEMÓRIA E CAPACIDADE DE ARMAZENAMENTO CONFORME A NECESSIDADE.
 
* REVISAR E CRIAR
  - CLIQUE EM "REVISAR + CRIAR".
  - APÓS A VALIDAÇÃO, CLIQUE EM "CRIAR".

* CONECTAR Á INSTÂNCIA
  - USE FERRAMENTAS COMO SQL SERVER MANAGEMENT STUDIO (SSMS) OU O AZURE DATA STUDIO.
  - UTILIZA O ENDPOINT DA INSTÂNCIA E AS CREDENCIAIS CONFIGURADAS PARA SE CONECTAR.
